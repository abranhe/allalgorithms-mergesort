## Install

```
npm install @allalgorithms/mergesort
```

## Usage Example

```js
const mergeSort = require('@allalgorithms/mergesort');

arr = [77, 2, 10, -2, 1, 7]

console.log(mergeSort(arr));
// -> [-2, 1, 2, 7, 10, 77]
```
